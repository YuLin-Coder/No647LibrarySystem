## 项目介绍
基于SSM框架的图书馆管理系统系统，该项目分为个角色。主要功能如下
基于Spring + Spring MVC + MyBatis的图书馆管理系统，使用Maven进行包管理。主要功能包括：图书查询、图书管理、图书编辑、读者管理、图书的借阅与归还以及借还日志记录等。
